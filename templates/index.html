Frequently Asked Questions

Please check the answer to some of the most asked questions before sending an email

  • Why are you creating websites if you are a GM at Pret with no prior experience in Software and Web Development?

    I was bored when placed on furlough in January so decided to teach myself programming. The best way to learn is to do, hence me coming up with this project.

  • Why in particular this project?

    Ever since I was promoted to GM in 2018, I have spent a few minutes every week browsing through the reviews our customers are leaving. For me, that is a broader way of capturing public views than just a targeted Mystery Shopper report.

  • What about the labor calculator?

    Every time we have changed the way we calculate profit(whether Gross or Shop Controllable) I would create a spreadsheet to make computing it faster.

  • Are you really planning on introducing suggestions based on review trends?

    That is the plan for now. I would need to learn Machine Learning and Pattern Recognition so I could write the algorithm from scratch. That will take a while.

  • Why are there only 5 reviews per shop?

    Google Developer accounts(like mine) are limited to pulling only 5 completely random(no sorting) reviews per branch. In order to see all reviews(in some cases hundreds), I would need a special access key associated with the business.

  • Will you improve on this website any further?

    Yes! One day, all the placeholder text, commented out code and further functionality plans will be finalized. I cannot provide an exact date since I will be returning to shop eminently.

  • Why can't I set up an account?

    While the database code is written for all potential passwords to be hashed and salted before saving, I am still not sure how I should be handling personal data(email) to adhere with GDPR. Since the whole project is free to use, I did not implement the register/login sections in this release.

  • Why are you listing shops that are no longer open?

    Instead of spending hours upon hours to manually build the shops database, it took me less than 10 minutes to write a web scraping script that took all the shops listed on https://locations.pret.co.uk/london. I could only eliminate duplicate entries and not permanently closed shops.
